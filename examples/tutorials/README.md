# Rust UdonSharp Tutorial Series

A comprehensive tutorial series that guides you through learning Rust UdonSharp development from basics to advanced topics.

## Tutorial Overview

### 01 - Basics
**[Getting Started with Rust UdonSharp](01-basics/)**
- Setting up your development environment
- Creating your first UdonSharp behavior
- Understanding the compilation pipeline
- Basic Unity integration

### 02 - Networking
**[Understanding VRChat Networking](02-networking/)**
- Network synchronization fundamentals
- Manual vs Continuous sync modes
- Ownership and master client concepts
- Network events and custom events

### 03 - UI Systems
**[Building User Interfaces](03-ui-systems/)**
- Unity UI integration
- Dynamic UI updates
- Event-driven UI patterns
- Responsive design for VR and desktop

### 04 - Performance
**[Performance Optimization](04-performance/)**
- VRChat performance constraints
- Memory management techniques
- Network optimization strategies
- Profiling and debugging tools

## Learning Path

**Complete Beginner**: Start with Tutorial 01, then work through the [hello-world](../hello-world/) example.

**Some Unity Experience**: Begin with Tutorial 01, then try [player-counter](../player-counter/).

**Experienced Developer**: Review Tutorial 01 quickly, then dive into Tutorial 02 and the [game-manager](../game-manager/) example.

**Performance Focus**: Complete Tutorials 01-02, then focus on Tutorial 04 and advanced examples.

## Prerequisites

Before starting the tutorials, ensure you have:
- Rust 1.70+ installed with `wasm32-unknown-unknown` target
- Unity 2022.3+ with VRChat SDK3 Worlds
- Basic understanding of programming concepts
- Familiarity with VRChat world creation (helpful but not required)

## Tutorial Structure

Each tutorial includes:
